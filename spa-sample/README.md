# Sample SPA

## Install dependencies

```sh
$ npm i
```

## Dev server

```sh
$ npm start
```

## Scenario document server

```sh
$ npm run start:scenario
```

## Build

```sh
$ npm run build
```

## Test

```sh
$ npm test
```

## Domain specific library document server

```sh
$ npm run start:doc
```

## Format

```sh
$ npm run format
```

## Lint and auto fix

```sh
$ npm run review
```
